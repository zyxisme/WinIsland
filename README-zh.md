# WinIsland

[English](./README.md) | 简体中文

> [!WARNING]
> 该项目仍在开发中 (WIP)，可能会出现错误。

这是一个可以在 Windows 上显示**灵动岛**的项目。

## 下载
你可以在 [Release](https://github.com/Eatgrapes/WinIsland/release) 或者我们的 [网站](https://tanikaze.icu/WinIsland) 下载WinIsland

## 构建项目

- **Rust** 环境
- **Cargo**

```cmd
git clone https://github.com/Eatgrapes/WinIsland.git

cd WinIsland

cargo build --release
```
