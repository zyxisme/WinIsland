## Download it
you can in [Release](https://github.com/Eatgrapes/WinIsland/release) or our [website](https://tanikaze.icu/WinIsland) Download WinIsland

## Build it 

- **Rust** environment
- **Cargo**

```cmd
git clone https://github.com/Eatgrapes/WinIsland.git

cd WinIsland

cargo build --release
```
